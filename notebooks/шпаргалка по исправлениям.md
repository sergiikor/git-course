# 🛠 Моя Шпаргалка по Git (Основы и исправление ошибок)

### 1. Исправление последнего коммита (Amend)
*Если забыл файл, добавил лишний или хочешь поправить текст.*

* **Изменить только текст:**
  `git commit --amend -m "Новый текст"`
* **Добавить забытый файл (без смены текста):**
  `git add .` + `git commit --amend --no-edit`
* **Удалить лишний файл из последнего коммита:**
  `git rm --cached файл` + `git commit --amend --no-edit`

---

### 2. Отмена изменений (Reset)
*Когда нужно откатиться во времени.*

* **Мягкая отмена (Soft):** Удаляет коммит, но оставляет все изменения в файлах (в индексе).
  `git reset --soft HEAD~1`
* **Жесткая отмена (Hard):** Полностью стирает последний коммит и все изменения в коде. **Опасно!**
  `git reset --hard HEAD~1`

---

### 3. Заначка (Stash)
*Если нужно срочно сменить задачу, но код еще не доделан.*

* **Спрятать изменения:** `git stash`
* **Посмотреть список заначек:** `git stash list`
* **Вернуть и удалить из заначки:** `git stash pop`

---

### 4. Работа с ветками (Rebase & Merge)
*Для поддержания чистоты истории.*

* **Rebase:** Переносит твою ветку на верхушку актуального `main`.
  `git rebase main`
* **Fast-forward Merge:** Склеивание веток в прямую линию (после ребейза).
  `git checkout main` -> `git merge feature-branch`

---

### 5. Полезные команды проверки
* **Короткий лог (последние 3 записи):** `git log --oneline -n 3`
* **Посмотреть файлы в последнем коммите:** `git show --name-only`
* **Увидеть разницу между коммитом и файлом:** `git diff`
* **Проверить, что в индексе (зеленое):** `git diff --cached`

---

### 💡 Полезный совет:
Никогда не делай `--amend` или `rebase` для коммитов, которые уже улетели на общий сервер (Push), если работаешь в команде. Это меняет историю!


### 6. Визуализация дерева (Графический лог)
*Чтобы понять, как переплетаются ветки.*

* **Красивое дерево в терминале:**
  `git log --oneline --graph --all`
  * `--graph` — рисует текстовую схему веток.
  * `--all` — показывает все ветки сразу (а не только текущую).
* **Супер-команда (с датами и авторами):**
  `git log --graph --abbrev-commit --decorate --format=format:'%C(bold blue)%h%C(reset) - %C(bold green)(%ar)%C(reset) %C(white)%s%C(reset) %C(dim white)- %an%C(reset)%C(bold yellow)%d%C(reset)' --all`
