## 📺 Структура курсу
1. Командний рядок – основи роботи з Git Bash.
2. Git – основні команди: ініціалізація, коміти, гілки, злиття.
3. GitHub – робота з віддаленим репозиторієм, пул-реквести, вирішення конфліктів.
